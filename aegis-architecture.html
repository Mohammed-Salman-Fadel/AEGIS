AEGIS Architecture

Local-First AI Assistant Platform · Rust Engine + React Frontend
v0.1.0 · 14K LOC Rust · 6K LOC TSX

Architecture Style

  • • Monolithic Rust engine with modular subsystems
  • • Separate Python RAG service (FastAPI)
  • • React SPA + Rust CLI as dual frontends
  • • MCP protocol for tool integrations
  • • Single-pass pipeline (not iterative agent loop)

Data Flow

  • • User → Frontend/CLI → HTTP/SSE → Router
  • • Router → Orchestrator::handle_fallback()
  • • Classify → RAG → Code Search → Zotero
  • • Prompt Synthesis → Inference Backend → Response
  • • Response streamed back + persisted to JSON

Key Technologies

  • • Rust (Tokio + Axum + reqwest + async-trait)
  • • React 18 + TypeScript + Vite
  • • Python FastAPI + Sentence-Transformers
  • • Ollama / LM Studio / OpenAI-compatible LLMs
  • • MCP (Model Context Protocol) for tools
